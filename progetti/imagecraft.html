Panoramica

ImageCraft è un’applicazione pensata per semplificare due operazioni molto comuni nella preparazione delle immagini: la rimozione dello sfondo e la creazione di icone Windows. Tutte le funzioni sono raccolte in un’unica interfaccia semplice, adatta sia all’utilizzo da PC sia da dispositivo Android.

Il programma permette di selezionare un’immagine, eliminare lo sfondo, controllare il risultato in anteprima e salvarlo in formato PNG con trasparenza reale. La stessa immagine può poi essere trasformata in un file .ico completo e compatibile con Windows.

Problema risolto

Per preparare un logo, una grafica o un’immagine destinata a un sito o a un programma è spesso necessario utilizzare strumenti diversi: uno per rimuovere lo sfondo, uno per sistemare le proporzioni e un altro per creare l’icona. ImageCraft riunisce queste operazioni e riduce i passaggi necessari, evitando conversioni manuali e semplici rinominazioni di file che non producono vere icone Windows.

Rimozione dello sfondo

Lo strumento analizza l’immagine e separa automaticamente il soggetto dallo sfondo. Il risultato viene mostrato sopra una trama a scacchi, così è possibile verificare immediatamente le aree trasparenti prima del salvataggio.

L’elaborazione può essere eseguita direttamente sul dispositivo senza inviare il file su Internet. Per immagini particolarmente difficili è disponibile anche una modalità online opzionale. Sono presenti controlli per migliorare i bordi, ammorbidire il ritaglio e ottenere un risultato più pulito sugli elementi sottili.

Esportazione PNG trasparente

Dopo la rimozione dello sfondo, l’immagine viene salvata in formato .png mantenendo il canale di trasparenza. Il file ottenuto può essere utilizzato in siti web, documenti, presentazioni, stampe, grafiche e altri programmi senza dover effettuare ulteriori conversioni.

Creazione di icone Windows

ImageCraft genera veri file .ico multi-risoluzione. Ogni icona contiene le dimensioni 16, 24, 32, 48, 64, 128 e 256 pixel, così Windows può scegliere automaticamente quella più adatta per Esplora file, collegamenti, finestre e barra delle applicazioni.

È possibile utilizzare l’immagine originale oppure quella appena elaborata senza sfondo. La modalità Immagine intera conserva tutte le proporzioni aggiungendo margini trasparenti, mentre la modalità Riempi icona adatta l’immagine al formato quadrato tramite ritaglio centrale.

Flusso di utilizzo

  1. Si seleziona un’immagine dal dispositivo.
  2. Si sceglie lo strumento per rimuovere lo sfondo o creare un’icona.
  3. Si regolano le opzioni disponibili in base al risultato desiderato.
  4. Si controlla l’anteprima dell’immagine elaborata.
  5. Si salva il PNG trasparente oppure il file ICO multi-risoluzione.

File supportati

Il programma può importare i formati immagine più comuni e produrre file PNG trasparenti e icone Windows ICO. I risultati vengono salvati direttamente nella posizione scelta dall’utente.

Interfaccia

L’interfaccia utilizza il tema Mannino Maker scuro con accenti verdi ed è organizzata in strumenti distinti. L’anteprima prima e dopo rende immediato il confronto, mentre i controlli principali restano sempre facilmente accessibili anche su schermi di dimensioni ridotte.

Versione 1.1.3

La versione 1.1.3 comprende la rimozione automatica dello sfondo, l’esportazione PNG trasparente, la creazione di icone ICO multi-risoluzione, il logo ImageCraft e la distribuzione tramite installer Windows e applicazione Android.

Destinatari

ImageCraft è pensato per maker, grafici, sviluppatori, studenti e utenti che hanno bisogno di preparare rapidamente immagini trasparenti, loghi e icone senza utilizzare più programmi separati.
